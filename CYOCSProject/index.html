Common CSS Properties

(and how to use them)

Where do I start?

When you begin learning CSS, the number of different properties that exist can be overwhelming. Here you will learn about some of the most important, foundational CSS properties and gain a good foundation for your continued learning.

Common CSS Properties and How to Use Them

In the table below you will find a breakdown of the most commonly used CSS properties. To learn more about a property, click on the name of the property or scroll down.

Property Uses
color Change the font color
font-family Change the font style
letter-spacing Change the letter spacing
opacity Change how visible something is

Color

The first and most commonly used CSS property is color. color can be used to change the color of the text within the html element you are targeting.

Font-Family

The second most commonly used CSS property is font-family. font-family can be used to change the font of the text. There are a wide variety of fonts to choose from, but its important to include back up fonts in case a particular font is unavailable to a user.

Letter-Spacing

letter-spacing is an important CSS property used to change the spacing between letters. This can be important to preserving the readability of a website when certain font styles or certain font families are used. It can also be used to emphasize specific letters, like in an acronym, or to stylize a word.

Opacity

opacity is a CSS property used to adjust the visibility or ability to see through an html element. It can be used to mix colors, overlap different images, and stylize a wepage. Opacity values are from 0-1, with 0 being completly invisible and 1 being completly visible.
